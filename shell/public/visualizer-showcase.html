04 Particle Dot Matrix Canvas

Grid of dots displaced along a sine wave, amplitude-modulated. Each particle has independent phase offset and size jitter. Creates a shimmering curtain of light. Dreamy and cinematic.

05 Gradient Glow Line Canvas

Single smooth bezier waveform with neon gradient stroke (cyan→indigo→violet). Multi-layered: thick blur pass + thin crisp line + trailing fade. More refined than SiriWave. Works perfectly for idle/thinking states.
