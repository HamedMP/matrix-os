# ACT 1: THE NOISE -- Image Generation Prompts

## Global Style Reference

**Visual language**: David Fincher color grading -- clinical, corporate, oppressive. Cold blue-white screen glow on skin and surfaces. Desaturated palette. Slightly overexposed highlights from screen light. Shallow depth of field. Tight framing. Photorealistic, cinematic, 35mm film grain. Every shot should feel like it was pulled from a $200M feature film.

**Consistent elements**: Cool blue (#1a2a4a to #3a5a8a) ambient light, harsh white screen glare, muted skin tones, dark shadows, modern but generic devices (no visible brand logos), 2024-era UI design language.

## Production Notes

### Consistency Checklist
- [ ] All shots use the same color temperature (cold blue, ~4000K screen glow)
- [ ] Skin tones are consistently desaturated and slightly sickly under screen light
- [ ] No identifiable brand logos on any device (phones, laptops, screens)
- [ ] UI text is in English, using generic sans-serif fonts (not platform-specific)
- [ ] Dark rooms throughout -- screens are the primary/only light source
- [ ] Consistent film grain level across all shots
- [ ] Shallow depth of field maintained (f/1.2 to f/2.8 equivalent)

### Seed/Style Consistency Tips
- Generate Shot 1.16 first (the wide shot) to establish the room, desk, and person
- Use that generation as a style reference for all close-up shots
- Maintain the same person's hands across shots 1.2, 1.3, 1.4, 1.5, 1.6, 1.14
- The phone should look the same across shots 1.1, 1.2, 1.9, 1.11, 1.15
- The laptop should look the same across shots 1.3, 1.4, 1.6, 1.7, 1.8, 1.12, 1.13, 1.14, 1.16
