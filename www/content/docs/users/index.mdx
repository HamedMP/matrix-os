Matrix OS is a personal cloud computer for AI-native development. You use it from `app.matrix-os.com`, from the `matrix` CLI, and from coding agents that run inside your Matrix workspace.

The first user path is intentionally developer-focused: create a Matrix computer, authenticate GitHub, clone a repo, launch a coding agent, and keep the work running without keeping your laptop open.

<Callout type="info" title="The workspace keeps running">
  Matrix sessions live on your Matrix computer. You can detach from a terminal, close your laptop, reconnect from the browser or CLI, and continue from the same shell state.
</Callout>

## Why Developers Use Matrix

- Your work runs on a dedicated cloud machine instead of the laptop in front of you.
- Full-stack dev servers, tests, previews, and agents can keep running while you switch devices.
- GitHub auth, source repos, terminal sessions, logs, transcripts, and app data live in one workspace.
- The web shell and CLI attach to the same persistent zellij-backed sessions.
- Mac and iOS apps are planned surfaces for the same Matrix computer, not separate workspaces.
